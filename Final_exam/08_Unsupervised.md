### 3) –ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ —è–¥—Ä–æ

**Agglomerative Clustering (bottom-up):**

   - –û–±–Ω–æ–≤–∏—Ç—å –º–∞—Ç—Ä–∏—Ü—É —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–π
3. –ü—Ä–æ–¥–æ–ª–∂–∞—Ç—å –¥–æ 1 –∫–ª–∞—Å—Ç–µ—Ä–∞

```

```

**2. Complete Linkage (maximum):**
```
d(C_i, C_j) = max{d(x, y) : x ‚àà C_i, y ‚àà C_j}
```
–ú–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ.

**–≠—Ñ—Ñ–µ–∫—Ç:** –∫–æ–º–ø–∞–∫—Ç–Ω—ã–µ, —Å—Ñ–µ—Ä–∏—á–µ—Å–∫–∏–µ –∫–ª–∞—Å—Ç–µ—Ä—ã.

**3. Average Linkage:**
```
d(C_i, C_j) = (1/(|C_i|¬∑|C_j|)) Œ£ Œ£ d(x, y)
                                 x‚ààC_i y‚ààC_j
```

**–ë–∞–ª–∞–Ω—Å** –º–µ–∂–¥—É single –∏ complete.

**4. Ward's Linkage:**
```
–û–±—ä–µ–¥–∏–Ω–∏—Ç—å –∫–ª–∞—Å—Ç–µ—Ä—ã, –º–∏–Ω–∏–º–∏–∑–∏—Ä—É—é—â–∏–µ –ø—Ä–∏—Ä–æ—Å—Ç inertia (–∫–∞–∫ –≤ K-means)
```

## 8.3 PCA (Principal Component Analysis)

### 1) –ò–Ω—Ç—É–∏—Ü–∏—è –∏ —Å—É—Ç—å

**–ü—Ä–æ—Å—Ç—ã–º–∏ —Å–ª–æ–≤–∞–º–∏ (ELI5):**

**–§–∏–∑–∏—á–µ—Å–∫–∏–π —Å–º—ã—Å–ª:**

### 2) –ó–∞—á–µ–º –∏ –∫–æ–≥–¥–∞ (Business Value)

**–ö–∞–∫—É—é –ø—Ä–æ–±–ª–µ–º—É —Ä–µ—à–∞–µ—Ç:**
- **Curse of dimensionality**: —Å–ª–∏—à–∫–æ–º –º–Ω–æ–≥–æ –ø—Ä–∏–∑–Ω–∞–∫–æ–≤
- **–í–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è**: –ø—Ä–æ–µ–∫—Ü–∏—è –Ω–∞ 2-3 –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã
- **Noise reduction**: –æ—Ç–±—Ä–∞—Å—ã–≤–∞–µ–º –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Å –º–∞–ª–æ–π –¥–∏—Å–ø–µ—Ä—Å–∏–µ–π (—à—É–º)
- **Feature engineering**: –≥–ª–∞–≤–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –∫–∞–∫ –Ω–æ–≤—ã–µ –ø—Ä–∏–∑–Ω–∞–∫–∏

**–ì–¥–µ —ç—Ç–æ –∏–Ω–¥—É—Å—Ç—Ä–∏–∞–ª—å–Ω—ã–π —Å—Ç–∞–Ω–¥–∞—Ä—Ç:**
- **EDA (Exploratory Data Analysis)**: –≤–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è –º–Ω–æ–≥–æ–º–µ—Ä–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö
- **Preprocessing**: –ø–µ—Ä–µ–¥ –∫–ª–∞—Å—Ç–µ—Ä–∏–∑–∞—Ü–∏–µ–π, –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ü–∏–µ–π
- **Computer Vision**: eigenfaces –¥–ª—è —Ä–∞—Å–ø–æ–∑–Ω–∞–≤–∞–Ω–∏—è –ª–∏—Ü
- **–ì–µ–Ω–æ–º–∏–∫–∞**: –∞–Ω–∞–ª–∏–∑ –≥–µ–Ω–Ω–æ–π —ç–∫—Å–ø—Ä–µ—Å—Å–∏–∏

- ‚ùå –ù—É–∂–Ω–∞ **–∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è** (–≥–ª–∞–≤–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã ‚Äî –ª–∏–Ω–µ–π–Ω—ã–µ –∫–æ–º–±–∏–Ω–∞—Ü–∏–∏, —Å–ª–æ–∂–Ω–æ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞—Ç—å)

### 3) –ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ —è–¥—Ä–æ

**–î–∞–Ω–Ω—ã–µ:** X (n √ó d), —Ü–µ–Ω—Ç—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ (—Å—Ä–µ–¥–Ω–µ–µ = 0 –ø–æ –∫–∞–∂–¥–æ–º—É –ø—Ä–∏–∑–Ω–∞–∫—É)

**–¶–µ–ª—å:**  
–ù–∞–π—Ç–∏ –æ—Ä—Ç–æ–≥–æ–Ω–∞–ª—å–Ω—ã–µ –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏—è (–≤–µ–∫—Ç–æ—Ä–∞) **w_1, w_2, ..., w_k**, –º–∞–∫—Å–∏–º–∏–∑–∏—Ä—É—é—â–∏–µ –¥–∏—Å–ø–µ—Ä—Å–∏—é –ø—Ä–æ–µ–∫—Ü–∏–π.

**–ü–µ—Ä–≤–∞—è –≥–ª–∞–≤–Ω–∞—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞:**
```

      ||w||=1           ||w||=1
```

–≠—Ç–æ **—Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–π –≤–µ–∫—Ç–æ—Ä** –º–∞—Ç—Ä–∏—Ü—ã –∫–æ–≤–∞—Ä–∏–∞—Ü–∏–∏ X^T X —Å **–Ω–∞–∏–±–æ–ª—å—à–∏–º —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º**.

**–ö–æ–≤–∞—Ä–∏–∞—Ü–∏–æ–Ω–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞:**

**SVD –ø–æ–¥—Ö–æ–¥ (–±–æ–ª–µ–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–π):**

- V ‚Äî –º–∞—Ç—Ä–∏—Ü–∞ –ø—Ä–∞–≤—ã—Ö —Å–∏–Ω–≥—É–ª—è—Ä–Ω—ã—Ö –≤–µ–∫—Ç–æ—Ä–æ–≤ (**–≥–ª–∞–≤–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã**)
- Œ£ ‚Äî –¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ —Å–∏–Ω–≥—É–ª—è—Ä–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π (—Å–≤—è–∑–∞–Ω—ã —Å –¥–∏—Å–ø–µ—Ä—Å–∏–µ–π)

**–ì–ª–∞–≤–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã:**

**Explained Variance Ratio:**
```

```

### K-means –ö–ª–∞—Å—Ç–µ—Ä–∏–∑–∞—Ü–∏—è –∏ Elbow Method

```python
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt

# –î–∞–Ω–Ω—ã–µ
X, y = make_blobs(n_samples=300, centers=4, random_state=42)
X_scaled = StandardScaler().fit_transform(X) # –í–°–ï–ì–î–ê –º–∞—Å—à—Ç–∞–±–∏—Ä—É–µ–º!

# 1. Elbow Method (–ø–æ–∏—Å–∫ K)
inertia = []
K_range = range(1, 10)

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42)
    kmeans.fit(X_scaled)
    inertia.append(kmeans.inertia_) # Sum of squared distances to center

# –ï—Å–ª–∏ —Å—Ç—Ä–æ–∏—Ç—å –≥—Ä–∞—Ñ–∏–∫, —Ç—É—Ç –±—É–¥–µ—Ç "–ª–æ–∫–æ—Ç—å" –Ω–∞ K=4
print("Inertia values:", [int(i) for i in inertia])

# 2. –§–∏–Ω–∞–ª—å–Ω–∞—è –∫–ª–∞—Å—Ç–µ—Ä–∏–∑–∞—Ü–∏—è
kmeans_final = KMeans(n_clusters=4, random_state=42)
labels = kmeans_final.fit_predict(X_scaled)
centers = kmeans_final.cluster_centers_

print("Labels shape:", labels.shape)
print("Centers shape:", centers.shape)
```
